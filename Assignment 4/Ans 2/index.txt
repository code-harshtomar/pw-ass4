HTML entities are special code that represent reserved characters, symbol or characters with specific meaning in html. They are used to display character that might otherwise be interpretend as HTML code or cause rendiring issues in web browsers.HTML entities are written using an ampersand(&) followed by a specific code or name, and ending with a semicolon(;).
5 commonly Used HTML Entities:

&lt;-Represents the less-than symbol<.
Example:<p> This is an example &lt;em&gt;text&lt;/em&gt;</P>

&gt; - Represents the greater-than symbol>.
Example:<p>For more information,visit our website &gt; <a href="#">here</a></p>

&amp; - Represents the ampersand symbol &.
Example: <p> For further details, contact us &amp; we'll assist you.</p>

&quot; - Represent a double quotation mark"
Example:<p> The title of the book is " The Great gatsby".</P>

&copy; - Represents the copyright symbol.
Example: <p> &copy;2023 ACME Corporation.All rights reserved.</p>
